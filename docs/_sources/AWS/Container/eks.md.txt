# EKS
## EKSのメリット
EKSの最大の特徴は、Kubernetesというオープンソースの特徴を活かして、周辺の行為範囲なコミュニティの再利用性。
さらに、ACK(AWS Contreollers for Kubernetes)を利用するとK8sとAWSの連携もシームレスになる。
