# データベースサービス
## Cloud SQL
Cloud SQLでは、MySQS、PostgreSQS、SQL Serverなどのリレーショナルデータベースを提供するマネージドサービス。
パッチの更新・適用、バックアップ管理など様々なメンテナンスを自動で提供してくれる。
単一リージョンでの利用がベース。

## Spanner
グローバルスケール対応のRDS。強整合と高可用性を利用率している。
1秒当たり数万回以上の読み込み書き込みがある場合に対応可能。

## Firestore
JSONライクKey-ValueのNoSQLデータ構造対応で、リアルタイムとスケーラビリティに優れる。

## Bigtable
超大型のデータに最適なNoSQL。時系列データやIoTデータなどのスループット重視に利用される。

## AlloyDB
AlloyDBはPostgresSQS互換の次世代リレーショナルデータベース。Cloud SQLと比較して高性能・高可用性に特化したフルマネージドのDBサービス。

|特性|内容|
|:----|:----|
|タイプ|リレーショナル（PostgreSQL 互換）|
|ストレージ|分離アーキテクチャ。計算ノードとストレージノードを分離して高速化|
|スケーラビリティ|高い（専用キャッシュや分析アクセラレーター搭載）|
|高可用性|高速フェイルオーバー、デフォルトでレプリケーション対応|
|分析対応|OLTP + OLAP ハイブリッドとしても使える（分析クエリも高速）|
|マイグレーション|Cloud SQL（PostgreSQL）や他のPostgreSQLからの移行が容易（互換性高）|
